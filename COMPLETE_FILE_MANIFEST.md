## 🎯 WHAT EACH FILE DOES

### page-lens-guide.liquid
Renders the Lens Guide page with 3 lens type cards explaining Single Vision, Progressive, and Blue Light lenses.

### page-size-guide.liquid
Renders the Size Guide page with frame size breakdown, size categories table, face shape guide, and measurement tips.

### page-prescription-guide.liquid
Renders the Prescription Guide page with 6 definition cards explaining prescription terms and includes a 5-item FAQ accordion.

### page-faq.liquid
Renders the FAQ page with 10 organized questions in 5 categories with smooth accordion interface.

### page-lens-guide.css
Styles the Lens Guide page with responsive card grid layout, proper spacing, colors, and hover effects.

### page-size-guide.css
Styles the Size Guide page with responsive table, card grid, proper mobile conversion, and professional spacing.

### page-prescription-guide.css
Styles the Prescription Guide page with definition cards, accordion styling, SVG image wrapper, and smooth animations.

### page-faq.css
Styles the FAQ page with accordion styling, smooth animations, responsive layout, and keyboard navigation support.

### page-faq.js
Provides accordion functionality for FAQ page including open/close logic, keyboard support, and URL hash support.

### page.lens-guide.json / page.size-guide.json / page.prescription-guide.json / page.faq.json
Template configuration files that link the Liquid sections to their respective page templates.
